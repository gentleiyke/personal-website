# Ikemefula Oriaku Personal Website

A modern personal portfolio website built with React and Vite. The site showcases my professional profile, skills, achievements, GitHub projects, blog content, and contact options.

## Features

- Responsive personal portfolio layout
- Home, About, Projects, Blog, and Contact pages
- GitHub-powered project listing
- Project search, filtering, sorting, and pagination
- Contact form with backend message storage
- Blog system with admin-protected post creation
- Google Analytics tracking
- SEO support with React Helmet
- Express and MongoDB backend API

## Tech Stack

### Frontend
- React
- Vite
- React Router
- Bootstrap
- React Helmet Async
- React GA4

### Backend
- Node.js
- Express
- MongoDB
- Mongoose
- JWT authentication
- Multer uploads
- dotenv
